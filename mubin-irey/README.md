# Wedding Invite Maroon Cream

A Next.js wedding invitation website designed for GitHub + Vercel deployment.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Deploy

1. Create a new GitHub repository.
2. Upload these files.
3. Push to GitHub.
4. Import the repository into Vercel.
5. Deploy.

## What to customize

- `components/client-page.tsx`
  - Couple names
  - Date and time
  - Address
  - Schedule
  - Gallery images
  - RSVP link
  - Map links

## Security-friendly free setup

- Keep the site mostly static.
- Use external RSVP forms.
- Do not put secrets in frontend files.
- Use external links for maps and forms.
